## SanityImage Component Contract

The only new UI component in this phase. Designed to be a drop-in replacement for `StrapiImage` at the call sites.

```
Props:
  image         TSanityImage    Required. Raw Sanity image object with asset._ref, hotspot, crop, alt.
  width         number          Required. Rendered pixel width passed to Next.js Image.
  height        number          Required. Rendered pixel height passed to Next.js Image.
  className     string          Optional. Forwarded to Next.js Image className.
  priority      boolean         Optional. Forwarded to Next.js Image priority. Default: false.

Internal behavior:
  - Builds URL via imageUrlBuilder(client).image(image).width(width).height(height).auto('format').url()
  - Applies hotspot: .focalPoint(hotspot.x, hotspot.y) when hotspot is present
  - Applies crop: .crop('focalpoint') when crop is present
  - Falls back to empty string alt if image.alt is undefined
  - Returns null if image.asset is undefined (no broken img placeholder)
```

Loading state: no skeleton or placeholder — matches existing `StrapiImage` behavior (Next.js handles blur placeholder via `placeholder="blur"` only if `blurDataURL` is provided; not introduced in this phase).

Error state: returns `null` silently when `image.asset` is missing — same behavior as current `getStrapiMedia()` returning null for missing assets.
