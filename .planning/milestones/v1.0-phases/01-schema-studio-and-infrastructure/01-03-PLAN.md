**SINGLETON DOCUMENTS (7 total — SCHEMA-01):**

Each page document has a `blocks` array field with per-page type restrictions (per D-08). Block field named `blocks` (per D-09).

**1. `home-page.ts`** — name: `"homePage"`, title: "Home Page"
Fields:
- `title` (string)
- `description` (text)
- `blocks` (array of [{ type: "heroSection" }, { type: "featuresSection" }])

**2. `contact-page.ts`** — name: `"contactPage"`, title: "Contact Page"
Fields:
- `title` (string)
- `subTitle` (string) — note: camelCase matches Strapi field name
- `description` (text)
- `blocks` (array of [{ type: "contactInfo" }, { type: "schedule" }, { type: "contactFormLabels" }, { type: "generalInquiryLabels" }])

**3. `gallery-page.ts`** — name: `"galleryPage"`, title: "Gallery"
Fields:
- `title` (string)
- `description` (text)
- `blocks` (array of [{ type: "photoGallery" }, { type: "heroSection" }])

**4. `training-page.ts`** — name: `"trainingPage"`, title: "Training"
Fields:
- `title` (string)
- `description` (string) — note: string type per Strapi schema (not text)
- `blocks` (array of [{ type: "ctaFeaturesSection" }, { type: "imageWithText" }])

**5. `training-programs-page.ts`** — name: `"trainingProgramsPage"`, title: "Training Programs Overview"
Fields:
- `title` (string)
- `description` (text)
- `trainingPrograms` (array of [{ type: "reference", to: [{ type: "trainingProgram" }] }], title: "Training Programs")
NOTE: No `blocks` field — Strapi uses a relation, not a dynamic zone. In Sanity, this is an array of references.

**6. `global-settings.ts`** — name: `"globalSettings"`, title: "Global Settings"
Fields:
- `title` (string)
- `description` (text)
- `header` (object: type: "header") — single inline object, NOT an array. Strapi has repeatable: false.
- `footer` (object: type: "footer") — single inline object, NOT an array.
NOTE: No `blocks` field — header and footer are direct component fields, not a dynamic zone.

**7. `main-menu.ts`** — name: `"mainMenu"`, title: "Navigation Menu"
Fields:
- `items` (array of [{ type: "menuDirectLink" }, { type: "menuDropdown" }], title: "Menu Items")
NOTE: This is the polymorphic array for SCHEMA-05. Strapi calls the field `MainMenuItems` but we use `items` for cleaner naming. Both `menuDirectLink` and `menuDropdown` are allowed, and Sanity uses `_type` discriminator automatically.

**COLLECTION DOCUMENTS (3 total — SCHEMA-02):**

**8. `training-program.ts`** — name: `"trainingProgram"`, title: "Training Program"

**9. `contact-form.ts`** — name: `"contactForm"`, title: "Training Contact Form"

- `clients` (number, validation: Rule => Rule.min(0)) — Strapi uses integer with min 0
- `info` (text) — not required per Strapi
